# Lambda Resize Image

This function will resize a image when it is uploaded on AWS S3, the main objective is to shrink the file size and the loading speed on App

# Configuration

```bash
# configure your aws-cli
$ aws configure

# create a env file
$ cp .env.example > .env
```

# Installation

```bash
# install all dependencies
$ npm i
```

# Deploy

```bash
$ npm run deploy
```
